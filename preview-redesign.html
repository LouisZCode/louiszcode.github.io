The Problem

Language learners plateau because generic apps lack personalized feedback and real conversational practice. Existing tools treat every learner the same, ignoring proficiency gaps and native language interference patterns.

The Approach

Built a full-stack platform that uses Claude's language understanding to generate adaptive lessons, provide contextual grammar corrections, and simulate natural conversations. Designed a multi-agent architecture where specialized agents handle vocabulary, grammar, and conversation separately.

The Result

A production application serving users across 4 languages (Spanish, German, Portuguese, English) with real-time speech-to-text, text-to-speech, and intelligent lesson progression. Deployed on Railway with PostgreSQL persistence.

Key Decisions

Chose Flask over Django for faster iteration. Used Minimax TTS for natural-sounding multilingual speech. Implemented session-based learning to track progress without requiring accounts. Prioritized conversation quality over feature breadth.
